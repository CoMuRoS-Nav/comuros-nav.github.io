Installation

Setup instructions for installing CoMuRoS-Nav

Prerequisites

  • ROS 2 Humble Hawksbill (LTS, recommended)
  • Ubuntu 22.04 (Jammy Jellyfish)
  • Python 3.10+ (default with Ubuntu 22.04)
  • Nav2 Navigation Stack
  • Git
  • colcon build tool

Verify that ROS 2 is installed and sourced correctly:

# Check ROS 2 environment variables
printenv | grep -i ROS

# Expected output should include at least:
# ROS_DISTRO=humble
# ROS_VERSION=2
# ROS_PYTHON_VERSION=3

If ROS_DISTRO=humble is present, your ROS 2 environment is correctly set up and ready for CoMuRoS-Nav.

Step 1: Clone CoMuRoS (Required Dependency)

CoMuRoS-Nav builds on top of the core CoMuRoS system. Clone it first:

cd ~/ros2_ws/src
git clone https://github.com/comuros/comuros.git

For more details on CoMuRoS, visit the CoMuRoS project page.

Step 2: Clone CoMuRoS-Nav Repositories

CoMuRoS-Nav consists of two separate packages. Clone both into your workspace:

cd ~/ros2_ws/src

# Navigation Manager
git clone https://github.com/comuros-nav/navigation_manager.git

# Path Planner
git clone https://github.com/comuros-nav/path_planner.git

Your src/ directory should now contain three folders: comuros/, navigation-manager/, and path-planner/.

Step 3: Install Dependencies

cd ~/ros2_ws
rosdep install --from-paths src --ignore-src -r -y

# Install Python dependencies for each package
pip install -r src/comuros/requirements.txt
pip install -r src/navigation-manager/requirements.txt
pip install -r src/path-planner/requirements.txt

Step 4: Build the Packages

cd ~/ros2_ws
colcon build --symlink-install
source install/setup.bash
echo "source ~/ros2_ws/install/setup.bash" >> ~/.bashrc

Step 5: Set API Keys

You need at least one API key (OpenAI, Gemini, or XAI/Grok). Add them temporarily:

export OPENAI_API_KEY="your_openai_key"
export GEMINI_API_KEY="your_gemini_key"
export XAI_API_KEY="your_grok_key"

Or permanently in ~/.bashrc.

CoMuRoS-Nav Tutorial

Complete guide to running the multi-robot navigation system for custom applications

Running the Complete System

CoMuRoS-Nav requires CoMuRoS running first, then the two CoMuRoS-Nav packages. Each component runs in its own terminal.

Terminal 1 - 4: Launch CoMuRoS

Start the full CoMuRoS system first (simulation, robot LLMs, control services, and chat interface). Refer to the CoMuRoS tutorial for the exact commands. CoMuRoS-Nav depends on CoMuRoS being fully up before proceeding.

Terminal 5: Navigation Manager

Runs the multi-robot navigation coordinator from the navigation-manager package:

ros2 run navigation_manager navigation_manager

What it does:

  • Subscribes to high-level navigation goals from CoMuRoS
  • Coordinates goal assignment across all active robots
  • Monitors robot states and handles goal completion callbacks

Terminal 6 - 8: Path Planner

The path-planner package can be launched two ways:

Option A - Launch file (recommended)

The launch file starts all three nodes together (path_request_reciever, path_planner_node, and path_writer) in a single command:

# Terminal 6: Launch all path planner nodes at once
ros2 launch path_planner path_planner_system.launch.py

Option B - Run nodes individually (3 terminals)

Use this for debugging or when you need control over each component separately:

# Terminal 6: Path writer - publishes computed paths
ros2 run path_planner path_writer

# Terminal 7: Path request receiver - listens for incoming goal requests
ros2 run path_planner path_request_reciever

# Terminal 8: Path planner node - core LLM-based planning logic
ros2 run path_planner path_planner_node

What each node does:

  • path_writer - publishes the planned paths to the navigation stack
  • path_request_reciever - receives goal requests from the navigation manager and queues them
  • path_planner_node - core node that uses the LLM to compute paths and trigger replanning on events

Example Commands

Once everything is running, command the navigation system through the CoMuRoS chat interface:

  • "Navigate all robots to the exit"
  • "Scout the corridor ahead"
  • "Form a line formation and move forward"
  • "Avoid the obstacle on the left and replan"
